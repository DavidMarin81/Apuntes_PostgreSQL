- [CREAR BBDD]()
	~~~~
 	CREATE DATABASE MiBaseDeDatos;
 	~~~~
- [BORRAR BBDD]()
 	~~~~
 	DROP DATABASE IF EXISTS MiBaseDeDatos;
 	~~~~

 - [CREAR TABLA]()
	~~~
	CREATE TABLE Alumno (
 		idAlumno INT NOT NULL,
 		nombre VARCHAR(20),
 		apellido VARCHAR(50)
 	);
	~~~
   	- *Para ver la tabla en PostgreSQL -> MiBaseDeDatos / Schemas / public / Tables / Alumno*
